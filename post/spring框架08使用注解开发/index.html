//等价于    <bean id="user" class="com.chun.pojo.User"/>
@Component
public class User {
    //相当于<property name = "name" value = "chun">
    @Value("chun")
    public String name ;
}

衍生的注解

  • @Component 有几个衍生注解，我们在web开发中，会按照MVC三层架构
    • @Repository dao层
    • @Service service层
    • @Controller controller层 这四个注解功能都是一样，代表某个类注册到Spring中，装配Bean

自动装配

  • @Autowired
  • @Nullable ：字段标记了这个注解，说明这个字段可以为Null
  • @Resource

作用域

  • @Scope(“singleton”) //prototype

xml与注解

  • xml： 更加万能，适用任何场合！维护方便简单
  • 注解：不是自己的类使用不了，维护相对复杂

xml与注解最佳实践：

  • xml用来管理bean
  • 注解只负责完成属性的注入

使用JAVA的方式配置Spring

我们现在要完全部使用Spring的xml配置，全部交给JAVA来做！ JAVAConfig是Spring的一个子项目，在Spring 4 之后， 它成为了一个核心功能
