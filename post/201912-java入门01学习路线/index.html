大概学习java路线：javase，数据库，前端，javaweb，SSM框架，Linux，SpringBoot，SpringCloud，Hadoop

JDK,JRE,JVM之间的关系

JDK(Java Development Kit):java开发工具包,JDK包含了JRE （可以独立安装） JRE(Java Runtime Environment):java程序的运行环境，JRE包括了JVM （可以独立安装） JVM(Java Virtual Machine):Java虚拟机，只能识别.class文件，将它解释为二进制（不能独立安装） 总结：JDK>JRE>JVM

JAVA 的加载与执行

JAVA程序运行：编译 与 运行

  1. 编译阶段
  • 检测.java源文件是否合法，合法则生成.class(字节码文件)。
  • 我们需要安装JDK（开发的工具包），使用里面的javac.exe(是一个工具，叫java编译器)，用这个工具把.java变成.class Tip:字节码文件不是纯粹的二进制文件，这种文件无法在操作系统当中执行
  1. 运行阶段
  • 使用JDK工具包里面的java.exe（负责运行的一个工具），然后会启动JVM（java虚拟机），JVM会启动类加载器（classloader）
  • classloader会去硬盘搜索.class文件,找到后把它装入JVM中
  • JVM将.class文件解释成二进制数据
  • 操作系统执行二进制和硬件平台进行交互
