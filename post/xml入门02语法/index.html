xml必须严格遵守语法

基本语法

  1. xml文档的后缀名 .xml
  2. xml文档第一行必须定义文档声明
  3. xml文档中有且仅有一个根标签
  4. 属性值必须使用引号（单双都可以）
  5. 标签必须正确关闭
  6. xml标签区分大小写
