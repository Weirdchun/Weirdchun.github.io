相关命令

启动

  • 启动服务：systemctl start mysqld.service

  • 关闭服务：systemctl stop mysqld.service

  • 登录本机的Mysql mysql -u root -p

远程登录

  • mysql允许远程登录： 
    1
2
3
4
5
6
7

    use mysql    

 #将host设置为%表示任何ip都能连接mysql，当然您也可以将host指定为某个ip
update user set host='%' where user='root' and host='localhost';

flush privileges;        #刷新权限表，使配置生效
  • mysql关闭远程登录： 
    1
2
3
4
5

     #将host设置为localhost表示只能本地连接mysql

 update user set host='localhost' where user='root';

 flush privileges;        #刷新权限表，使配置生效

备注：您也可以添加一个用户名为yuancheng，密码为123456，权限为%（表示任意ip都能连接）的远程连接用户。命令参考如下：

1
2

 grant all on *.* to 'yuancheng'@'%' identified by '123456';
 flush privileges;

当SQLyog连接mysql出现错误2058"plugin caching_sha2_password could not be loaded"时，解决办法

1
2
3

ALTER USER 'root'@'localhost' IDENTIFIED BY 'password' PASSWORD EXPIRE NEVER; #修改加密规则 

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'password'; #更新一下用户的密码
