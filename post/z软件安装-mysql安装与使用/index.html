很多教程都是Mysql的，所以我也先学习mysql吧

第一次使用MySQL

Mysql原理

其实我们安装的MySQL是一个服务，别人可以通过访问我们电脑的IP与端口，来访问MySQL

  1. MySQL目录结构 数据库：文件夹 表：文件 数据：文件里面的数据

安装与配置

在网上各种途径下载mysql8.0.18.zip（Mysql压缩包）,把它压缩到你想要的位置（其实就是安装位置了）。 我把它压缩到了G:\mysql\mysql-8.0.18-winx64然后在这个文件下

  • 创建my.ini配置文件，编辑配置信息

[client]
# 设置mysql客户端默认字符集
default-character-set=utf8
 
[mysqld]
# 设置3306端口
port = 3306
# 设置mysql的安装目录
basedir=G:\mysql\mysql-8.0.18-winx64

//下面我没有配置
# 设置 mysql数据库的数据的存放目录，MySQL 8+ 不需要以下配置，系统自己生成即可，否则有可能报错
# datadir=C:\\web\\sqldata
# 允许最大连接数
max_connections=20
# 服务端使用的字符集默认为8比特编码的latin1字符集
character-set-server=utf8
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
  • 加入到环境变量 把G:\mysql\mysql-8.0.18-winx64\bin加入到系统变量的PATH中，才能使用下面启动mysql的命名

启动

  • 管理员身份打开cmd(我用的VScode)
1
2

PS C:\> cd G:\mysql\mysql-8.0.18-winx64\bin //进入到bin文件
PS G:\mysql\mysql-8.0.18-winx64\bin>
  • 初始化数据库 mysqld --initialize --console
1
2
3
4

PS G:\mysql\mysql-8.0.18-winx64\bin> mysqld --initialize --console
...
A temporary password is generated for root@localhost: APWCY5ws&hjQ //得到一个初始密码，后面登录会使用上
...
  • 输入安装命令 mysqld install
1
2
3

PS G:\mysql\mysql-8.0.18-winx64\bin> mysqld install
The service already exists!
The current server installed: G:\mysql\mysql-8.0.18-winx64\bin\mysqld.exe MySQL
  • 输入启动命名 net start mysql
1
2
3
4

PS G:\mysql\mysql-8.0.18-winx64\bin> net start mysql
请求的服务已经启动。

请键入 NET HELPMSG 2182 以获得更多的帮助。

登录

登录本机的Mysql mysql -u root -p

1
2
3
4
5

PS G:\mysql\mysql-8.0.18-winx64\bin> mysql -u root -p
Enter password: *****   //输入初始密码
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 15
mysql>

Tip：登录失败注意可能是my.ini的编码格式不对，必须是ANSI编码

改密码

初始密码太难记了，可以自己修改密码

1
2

PS G:\mysql\mysql-8.0.18-winx64\bin> mysqladmin -u root -p password 输入新密码
Enter password: 输入初始密码

退出

退出数据库exit或者 quit

1
2
3

mysql> exit
Bye
PS G:\mysql\mysql-8.0.18-winx64\bin>
