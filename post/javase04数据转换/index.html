数据转换分为两种：强制类型转换和自动类型转换 数据类型转换优先级由低到高： byte -> short -> char -> int -> long -> float -> double (先是小数点型大于整数型，然后是按字节大小)

数据转换

  1. 不能对布尔值进行转换
  2. 不能把对象类型转换为不相干的
  3. 高优先级到低优先级，强制转换
  4. 转换的时候可能存在内存溢出，或者精度问题
  • 内存溢出

2

      int i=128;
  byte b =(byte)i; 
  System.out.println(b);//此时输出的是一个随机数，因为"128"已经超过了byte类型
