安装软件一般有三种方式：

  • rpm ：用JDK演示，在线发布一个springboot项目
  • 解压缩：用tomca演示，启动并通过外网访问，发布网站
  • yum在线安装：用docker演示，直接安装运行跑起来docker就行

rpm JDK安装

开发JAVA程序必须要的环境！

  1. 下载JDK rpm包（jdk-8u241-linux-x64.rpm） rpm下载地址http://www.oracle.com/technetwork/java/javase/downloads/index.html

  2. 安装java环境

    • 检测当前系统是否存在java环境

      1

      # java -version

    • 如果有的话就卸载

      1
2
3
4

      # 检测JDK版本，列出所有被安装的rpm package
# rpm -qa|grep jdk
# 强制移除
# rpm -e --nodeps jdk_x

    • 安装

      1
2
3

      # 安装rpm包，其中i表示安装，v表示显示安装过程，h表示显示进度
# # rpm -ivh jdk-8u241-linux-x64.rpm

    • 配置环境变量

      在文件的最后面 vim /etc/profile添加java的配置

      1
2
3
4

      JAVA_HOME=/usr/java/jdk1.8.0_241-amd64
CLASSPATH=%JAVA_HOME%/lib;%JAVA_HOME%/jre/lib
PATH=$JAVA_HOME/bin;$JAVA_HOME/jre/bin
export PATH CLASSPATH JAVA_HOME

      让这个配置文件生效！source /etc/profile

  3. 项目发布 IDEA中把springboot项目生成的 jar包，放到服务器/home/chun中 然后在服务器中执行java -jar xxxx.jar ，然后就在服务器中跑起来了。 打开浏览器输入地址：http://182.92.87.108:9000/ 即可访问

# 查看firewall服务状态
systemctl status firewalld

# 开启、重启、关闭、firewalld.service服务
# 开启
service firewalld start
# 重启
service firewalld restart
# 关闭
service firewalld stop

# 查看防火墙规则
firewall-cmd --list-all    # 查看全部信息
firewall-cmd --list-ports  # 只看端口信息

# 开启端口
开端口命令：firewall-cmd --zone=public --add-port=80/tcp --permanent
重启防火墙：systemctl restart firewalld.service

命令含义：
--zone #作用域
--add-port=80/tcp  #添加端口，格式为：端口/通讯协议
--permanent   #永久生效，没有此参数重启后失效

安装Docker(yum在线安装)

基于 CentOS 7 安装

1.官网安装参考手册：https://docs.docker.com/install/linux/docker-ce/centos/

2.确定你是CentOS7及以上版本

1
2

[root@192 Desktop]# cat /etc/redhat-release
CentOS Linux release 7.2.1511 (Core)

3.yum安装gcc相关（需要确保 虚拟机可以上外网 ）

1
2

yum -y install gcc
yum -y install gcc-c++

4.卸载旧版本

1
2
3
4
5
6
7
8
9

yum -y remove docker docker-common docker-selinux docker-engine
# 官网版本
yum remove docker \
          docker-client \
          docker-client-latest \
          docker-common \
          docker-latest \
          docker-latest-logrotate \
          docker-logrotate \

5.安装需要的软件包

1

yum install -y yum-utils device-mapper-persistent-data lvm2

6.设置stable镜像仓库

1
2

4
5
6

1
