增加(Create)、读取(Retrieve)、更新(Update)和删除(Delete) 总体思路：

  1. 编写接口
  2. 编写对应的mapper中的sql语句
  3. 测试

select

<select id="getUserList" resultType="com.chun.pojo.User">

  • id:对应namespace中的方法名
  • resultType:sql语句执行的返回值！
  • parameterType:参数类型

实现：

  1. 编写接口

    select * from mybatis.user where id = #{id}
</select>
</mapper>

万能Map

如果我们的实体类，或者数据库中的表，字段或参数过多，考虑使用Map！ 区别：

  • Map传递参数，直接在sql中取出key parameterType="map"
  • 对象传递参数，直接在sql中取对象的属性 parameterTyper="Object"
  1. UserMapper
1
2

    //万能Map <String>表示key，<Object>表示value
    int addUser2(Map<String,Object> map);
  1. UserMapper.xml
1
2
3
4

<!--    传的是map的key-->
    <insert id="addUser2" parameterType="map" >
        insert into mybatis.user(id,name,pwd) value (#{userid},#{userName},#{passWord})
    </insert>
