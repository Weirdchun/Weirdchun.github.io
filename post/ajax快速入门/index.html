ajax快速入门

2020-03-13
ajax
times read

Contents

  • 历史

AJAX = Asynchronous JavaScript and XML（异步的 JavaScript 和 XML）。 AJAX 不是新的编程语言，而是一种使用现有标准的新方法。 AJAX 是与服务器交换数据并更新部分网页的艺术，在不重新加载整个页面的情况下。

历史

传统的网页，要更新内容或提交表单，需要重新加载整个页面， 而使用ajax技术的网页，可以在后台服务器进行少量的数据交换，实现异步局部更新 例子：

  1. 在百度搜素框中，输入一个“为”，服务器会返回“为什么”。
  2. 注册时，输入用户名自动检测用户是否存在
